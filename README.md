## About
twitter-app : it is an app built with React to emulate twitter web app .

## Scenarios 

- login
- sign up
- log out
- reset password 
- add tweet
- remove tweet
- add reply
- remove reply
- add like ❤️
- remove like ❤️
- add avatar
- add cover 
- search 
 
> PS: this app is not stable and it's only for preview
 
## Installation and Run

- back-end
```sh
npm i
npm start
``` 
- front-end 
```sh
cd client
npm i
npm start
```
